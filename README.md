# Qazaq Rehab - Digital Rehabilitation Center

## Project Overview
Qazaq Rehab is Kazakhstan's first digital rehabilitation center offering innovative healthcare solutions using cutting-edge technology. This web application provides a comprehensive platform for patients to access rehabilitation services, book consultations, and interact with medical professionals.

## Setup Instructions

### Prerequisites
- Python 3.7+
- PostgreSQL database
- Modern web browser

### Installation
1. Clone the repository
2. Install Python dependencies: `pip install flask flask-cors psycopg2 google-generativeai`
3. Set up PostgreSQL database with required tables
4. Configure database connection in `server.py`
5. Run the Flask server: `python server.py`
6. Open `HomePage/index.html` in a web browser
